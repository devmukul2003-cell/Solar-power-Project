# Quick Start Guide

## 🚀 Getting Started in 3 Steps

### Step 1: Open the Website
Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

### Step 2: Create Account or Login
- Click "Get Started" on the home page
- Use demo credentials:
  - Email: `demo@energy.com`
  - Password: `demo123`
- Or create a new account with your details

### Step 3: Explore the Dashboard
- View real-time energy statistics
- Interact with charts
- Change date ranges and view types
- Export your data
- Check energy-saving tips

## 📊 Dashboard Features

### Statistics Cards
- **Solar Energy Produced**: Daily average based on selected range
- **Energy Consumed**: Total consumption including grid power
- **Money Saved**: Calculated at ₹8 per kWh
- **CO₂ Reduced**: Environmental impact (0.82 kg per kWh)

### Interactive Filters
- **Date Range**: 7, 30, 90, or 100 days
- **View Type**: Daily, Weekly, or Monthly aggregation
- **Export**: Download data as CSV file

### Charts
- **Hourly Chart**: Today's generation vs consumption
- **Overview Chart**: Historical data based on filters

## 💡 Energy Tips Page

Browse personalized recommendations:
- High Impact tips (₹500-800/month savings)

## 🐛 Troubleshooting

**Charts not showing?**
- Check internet connection (Chart.js loads from CDN)
- Refresh the page

**Login not working?**
- Use exact demo credentials
- Check browser console for errors

**Data not persisting?**
- Enable cookies/localStorage in browser
- Don't use incognito mode
