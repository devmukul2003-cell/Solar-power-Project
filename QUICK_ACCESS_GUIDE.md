## ⚡ Feature Quick Reference

### 1. Incentive Points ⭐
**Location**: Yellow card below stats  
**Formula**: 10 points per kWh solar used  
**Actions**: View total, earned, redeemed | Reset points

### 2. Language Toggle 🌐
**Location**: Top right navigation  
**Options**: English (EN) ↔ Hindi (हिं)  
**Effect**: All UI text updates instantly

### 3. Theme Toggle 🌙☀️
**Location**: Top right navigation  
**Options**: Light Mode ↔ Dark Mode  
**Effect**: Colors and backgrounds change

### 4. Date Range 📅
**Location**: Filters section  
**Options**: 7, 30, 90, 100 days, Custom  
**Effect**: Filters all data and charts

## 🔍 Troubleshooting

### Issue: Features not working
**Solution**: 
1. Clear browser cache
2. Refresh page (Ctrl + F5)
3. Check browser console (F12)
4. Try incognito mode

### Issue: Data not loading
**Solution**:
1. Check internet connection (for Chart.js)
2. Verify localStorage is enabled
3. Try demo account login
4. Check browser compatibility

### Issue: Points showing 0
**Solution**:
1. Wait for data to load (5 seconds)
2. Check if energy data exists
3. Refresh the page
4. Check browser console for errors

## 📊 Data Insights

### Understanding Your Points
```
Example Calculation:
Solar Generated: 10 kWh
Consumed: 15 kWh
Solar Used: min(10, 15) = 10 kWh
Points Earned: 10 × 10 = 100 points
```

### Reading the Stats
- **Solar Produced**: Energy from your panels
- **Energy Consumed**: Total household usage
- **Money Saved**: ₹8 per kWh from solar
- **CO₂ Reduced**: 0.82 kg per kWh solar
